# GeneticCarLearner

NEAT-based autonomous driving experiment project.

## Overview

This repository is the public container for the Genetic Car Learner project referenced in my portfolio. The project goal is to evolve a neural network controller for driving behavior using NEAT (NeuroEvolution of Augmenting Topologies).

## Current Repository State

- The current repo snapshot contains the NEAT feedforward configuration file: `condig-feedforward.txt`.
- Source code and training/visualization scripts are not included in this snapshot.
